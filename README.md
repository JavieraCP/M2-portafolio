# M2-portafolio
Este proyecto es una plantilla básica de una tienda de comercio electrónico (E-Commerce) construida con HTML y CSS. Sirve como punto de partida para una aplicación web más completa de venta de productos.
El objetivo de este proyecto es simular la estructura principal de una página de productos de una tienda online. Incluye:
Una barra de navegación (<nav>) con enlaces principales.
Una sección de productos (<section class="productos">) que muestra un listado en formato de cuadrícula (<div class="grid">).
Tarjetas de producto (<div class="card">) con imagen, título, precio y botones de acción.
Un pie de página (<footer>) con información de derechos de autor.

Tecnologías utilizadas:
HTML5: Utilizado para la estructura y el contenido de la página.
CSS: Utilizado para el diseño, layout y estilos visuales de la página.
Javascript: Utilizado para añadir algunas funciones interactivas a la página
